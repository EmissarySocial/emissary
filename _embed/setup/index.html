Welcome To Emissary

This is the Emissary setup console! It helps administrators configure the domains that Emissary will serve, along with other file locations and settings.
Visit the Emissary documentation for more help.
IMPORTANT: This app is not password protected, so it should only be run from a private, trusted machine, and NOT from a publicly available server. Once you have configured your Emissary instance, you can turn off the console. It does not need to be running for Emissary to work.
